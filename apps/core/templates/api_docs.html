API Documentation

Comprehensive guide to YAZ Healthcare Platform APIs

Overview

The YAZ Healthcare Platform provides RESTful APIs for all healthcare applications. All APIs use JSON format and follow standard HTTP conventions.

Base URL

http://localhost:8000

Response Format

JSON

Authentication

Bearer Token

Platform APIs

Health Check

GET
/health

Check platform health and status

Dashboard Statistics

/api/v1/dashboard/stats

Get dashboard statistics and metrics

Recent Activity
